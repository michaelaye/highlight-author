# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2026-03-03

### Added

- Zenodo DOI badge in README.
- `CITATION.cff` for standardized citation metadata.
- `CHANGELOG.md` following Keep a Changelog format.
- "Related tools" section in README linking to [chronobib](https://github.com/michaelaye/chronobib) and [ads-bib-tools](https://github.com/michaelaye/ads-bib-tools).

## [0.1.0] - 2026-03-03

### Added

- Initial release.
- Highlight a specific author's surname (and given name parts) in bibliography entries.
- Configurable styles: bold (default), italic, underline, or custom CSS class.
- Shorthand and full-form YAML configuration.
- Runs citeproc internally — compatible with other filters that need post-citeproc processing.
